The following code is a simple file manager for the EffortLogger System.
samplelog.txt contains a potential log file format.

Things you can do:
1. Create blank log file
2. Combine two log files (with the option of removing sensetive data)
3. Append to log file using the appendToExistingLogFile function

Format:
All log file must follow the format of rows being seperated by the \n
and columns seperated by two tabs \t\t. Every file has a header which can
be modified in the logFiles class.
